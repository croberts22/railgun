This is the unofficial API for MyAnimeList, the best site there is to keep track of the anime you've watched or plan to watch and the manga you've read. It's like Facebook for anime and manga lovers.

This API is a supplement to the official API that's currently being developed. Much development of this API has been stalled as of recently, and so I've taken the challenge to improve it. Read the API documentation for more information.

For announcements and updates, follow @spacepyro on Twitter.
